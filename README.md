# Da Chat

Simple and easy-to-use chat app built on JavaScript and PHP


## Built With

* [Bulma](http://bulma.io/) - The CSS framework used
* [jQuery](https://jquery.com/) - JS Library
* [PHP](http://www.php.net/) - Used for back-end
